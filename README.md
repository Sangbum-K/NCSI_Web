# NCSI_Web
NCSI Web Pages Renewal


---

## 프로젝트 목표
- 웹 사용자 경험 개선
  - 기존 지수 데이터의 가독성 및 접근성 향상
  - 복잡한 점수 체계를 직관적으로 탐색 가능한 UI 설계
  - 사용자의 데이터 접근 경로 최적화 및 편의성 강화

- 데이터 시각화 및 고도화
  - Power BI 연동을 통한 실시간 대시보드 및 분석 환경 구축
  - 정적인 테이블 위주의 데이터를 동적 차트 및 그래프로 전환
  - 실시간 데이터 연동을 통한 최신 지수 정보 제공
